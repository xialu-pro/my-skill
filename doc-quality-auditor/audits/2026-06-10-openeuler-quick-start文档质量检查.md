## 6. Top 5 快速修复项

### 🔴 1. 添加开篇摘要（C02 直接回答）

在第1行添加要点汇总框：

> openEuler可通过6步快速安装：①准备硬件 ②获取ISO ③校验完整性 ④启动安装 ⑤执行安装 ⑥查看系统信息。最低配置：4GB内存、120GB硬盘。预计耗时30-60分钟。

### 🔴 2. 添加FAQ章节（C09 FAQ覆盖）

在文末添加至少5个常见问题：
- 安装失败如何回退？
- 虚拟机安装步骤？
- 启动顺序设置错误怎么办？
- 分区方案建议？
- 网络配置方法？

### 🔴 3. 添加术语表（C04 定义先行）

iBMC（智能基板管理控制器）、RAID（独立磁盘冗余阵列）、LOM（板载网卡）、LVM（逻辑卷管理）首次出现时添加解释或链接。

### 🟡 4. 添加ISO选择决策框（C08 用例映射）

添加决策指引：
- 服务器完整部署 → Offline Everything ISO
- 最小化安装 → Offline Standard ISO
- 网络安装 → Network Install ISO

### 🟡 5. 添加操作验证步骤（Exp10 局限声明）

关键操作后添加验证命令+预期输出，如：
```bash
cat /etc/os-release
# 预期输出包含: 24.03 LTS SP3
```
